ICT Fundamentals

Computer Basics

Computer basics refer to the fundamental understanding of how computers work. It includes knowing the main components such as hardware, software, input, processing, storage, and output.

Input Devices

Input devices allow users to send data to a computer. Examples include keyboards, mouse, scanners, microphones, and webcams which help enter information into the system.

Output Devices

Output devices present information from the computer to the user. Examples include monitors, printers, speakers, and projectors. These devices display the processed data.

Storage and Memory

Storage devices save data permanently or temporarily. Examples include hard drives, SSDs, USB drives, and RAM. Memory helps the computer access information quickly.

Software Types

Software is divided into system software and application software. System software manages the computer hardware while application software allows users to perform specific tasks like word processing.

Networking and Internet

Networking connects multiple computers to share data and resources. The internet is the largest network that allows global communication and access to online services.

Emerging ICT Technologies

Emerging technologies include Artificial Intelligence, Cloud Computing, Internet of Things (IoT), and Virtual Reality. These innovations are changing how people interact with technology.
