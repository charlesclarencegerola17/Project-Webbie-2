System Core Components

Bootloader

A bootloader is a program that loads the operating system when a computer starts. It prepares the system hardware and then transfers control to the operating system kernel.

Power Supply Unit

The Power Supply Unit (PSU) converts electrical power from an outlet into usable power for computer components such as the motherboard, CPU, and graphics card.

Web Browser

A web browser is software used to access and view websites on the internet. Examples include Google Chrome, Microsoft Edge, Firefox, and Safari.

External Storage Devices

External storage devices are used to store and transfer data outside the computer. Examples include USB flash drives, external hard drives, and memory cards.

BIOS / UEFI Settings

BIOS or UEFI is firmware stored on the motherboard that initializes hardware during startup and allows users to configure system settings such as boot order and hardware configuration.
